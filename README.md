# Estela Duarte — Psicóloga Clínica

Site profissional para consultório de psicologia com foco em design elegante e experiência do usuário.

## Desenvolvedor

**Lucas Kikuthi**

## Sobre o Projeto

Site institucional desenvolvido para apresentar os serviços de psicoterapia da profissional Estela Duarte. O projeto foi construído com foco em:

- Design sofisticado e minimalista
- Experiência do usuário intuitiva
- Performance otimizada
- Responsividade total
- Acessibilidade

## Tecnologias Utilizadas

- **React 18** - Biblioteca JavaScript para interfaces
- **TypeScript** - Tipagem estática
- **Vite** - Build tool e dev server
- **Tailwind CSS** - Framework CSS utility-first
- **Framer Motion** - Animações fluidas
- **React Router** - Navegação entre páginas
- **shadcn/ui** - Componentes UI reutilizáveis
- **Lucide React** - Ícones modernos

## Estrutura do Projeto

```
src/
├── assets/          # Imagens e recursos estáticos
├── components/      # Componentes reutilizáveis
│   ├── ui/         # Componentes da biblioteca shadcn/ui
│   ├── Layout.tsx  # Layout principal
│   ├── Navbar.tsx  # Navegação
│   └── Footer.tsx  # Rodapé
├── pages/          # Páginas da aplicação
├── hooks/          # Hooks customizados
├── lib/            # Utilitários
└── index.css       # Estilos globais
```
